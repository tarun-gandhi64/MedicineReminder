# MedicineReminder
A full-stack web application that helps users manage medicine schedules, receive reminder notifications, track medicine history, and get basic health recommendations.
# Features
Add/Edit/Delete medicine reminders

Browser notification alerts

Snooze reminders

Mark medicine as taken

Medicine history tracking

Refill alerts

Basic health recommendation system
# Tech Stack
Frontend: HTML, CSS, JavaScript

Backend: Node.js, Express.js

Database: MongoDB, Mongoose
# Installation
git clone https://github.com/tarun-gandhi64/MedicineReminderApp.git

cd MedicineReminderApp/backend

npm install

node server.js

http://localhost:5000
